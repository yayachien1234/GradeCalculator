# 成績計算器

這是一個成績計算系統，設計用於幫助學生追蹤和管理各科目成績。使用者可以新增、刪除、更新科目，並列印成績單以便計算 GPA 和總平均。

## 功能

1. **新增科目** - 在成績單上新增科目的資料，包含科目代碼、分數和學分數。
   - 指令格式：`create 科目代碼 分數 學分數`
   - 範例：`create F7-001 82 2`

2. **刪除科目** - 從成績單中刪除科目。
   - 指令格式：`delete 科目代碼`
   - 範例：`delete F7-001`

3. **更新科目** - 更新已存在科目的分數或學分數。
   - 指令格式：`update 科目代碼 分數 學分數`
   - 範例：`update F7-001 75 4`

4. **列印成績單** - 列印所有科目、分數、等第、學分數，並計算總平均和 GPA。
   - 指令格式：`print`

5. **退出選單** - 結束程式。
   - 指令格式：`exit`

## 錯誤處理

- 不符合格式的指令會顯示錯誤訊息。
- 成績或學分數超出範圍時會提示錯誤訊息。
- 若試圖更新或刪除不存在的科目，系統會提醒科目不存在。
